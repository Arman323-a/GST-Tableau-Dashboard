
#  GST Collection & Growth Insights Dashboard

##  Project Overview
This project presents an interactive Business Intelligence dashboard analyzing GST collection trends across Indian states. The goal is to uncover revenue patterns, taxpayer contribution, and state-wise performance.

##  Problem Statement
Lack of centralized visibility into GST revenue trends, growth patterns, and state-wise contributions made it difficult to derive actionable insights.

## Approach
- Processed multi-year GST dataset (2018–2025)
- Designed and developed dashboard using Tableau
- Created key KPIs:
  - Total GST Collection
  - GST per Taxpayer
  - Year-over-Year Growth
- Built interactive visualizations:
  - State-wise contribution map
  - Import & Export distribution
  - Target vs Actual performance
- Added filters for dynamic analysis (State, Month)

##  Key Insights
- Top 3 states contribute ~40–50% of total GST revenue
- Consistent growth trend observed post-2020
- Significant variation in taxpayer contribution across states
- High-performing states include Tamil Nadu, Gujarat, and Karnataka
